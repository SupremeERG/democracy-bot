# Democracy-Bot Roadmap

# Goal
Create a discord democracy where users vote and elect roles with a discord bot that utilizes smart contracts to handle election data.

# Features

- Create election for role with an end time
- join candidacy
- vote for candidates
- end election prematurely
- get election stats
- end election automatically after end time
- automatically replace previously elected officials after new election if wanted
- User chooses dedicated election channel where elections/results are announced
